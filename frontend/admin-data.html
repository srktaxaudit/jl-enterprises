Import / Export

Export the books for your CA / Tally, take a backup, or bulk-import ledgers.

Export

CSV/Tally use the From–To range (Documents also the type); Trial Balance uses “As on”. Backup exports everything.

Import ledgers (Chart of Accounts)

CSV columns: code, name, type, group, openingBalance, openingSide, gstRate, gstin, hsnCode. Type = ASSET / LIABILITY / EQUITY / INCOME / EXPENSE; side = DR / CR. A header row is optional. Existing codes are skipped.
